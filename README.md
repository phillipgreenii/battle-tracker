Tabletop battle tracker


## Quick Start

Install Node.js and then:

```sh
$ git clone git@github.com:phillipgreenii/battle-tracker.git
$ cd battle-tracker
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt develop or grunt run
```
